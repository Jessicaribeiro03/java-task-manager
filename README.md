# Java Task Manager

## Description
A simple task management system developed in Java.  
It features an interactive terminal menu, allowing users to add, complete, and remove tasks, with automatic saving to a file for persistence.

## Features
- Add new tasks
- List tasks with status (pending or completed)
- Mark tasks as completed
- Remove tasks
- File persistence using a text file

## Technologies
- Java
- Eclipse IDE
- ArrayList
- Scanner
- File I/O (BufferedReader / BufferedWriter)
